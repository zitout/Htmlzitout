# مولد الأرقام العشوائية

هذا المشروع عبارة عن صفحة ويب بسيطة تقوم بتوليد رقم عشوائي مكون من 5 أرقام كل 5 دقائق.

## الميزات
- توليد رقم عشوائي من 5 أرقام
- تحديث تلقائي كل 5 دقائق
- تصميم بسيط وسهل القراءة
